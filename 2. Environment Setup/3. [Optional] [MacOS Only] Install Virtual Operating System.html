Installing Virtual Machine (MAC)


If you want to use a virtual machine with a Mac computer, you may do so by the following option:

1. Install VMWare Fusion (which at the time of this writing has a $80 license fee), and follow along with the previous video

2. Install VirtualBox (which is a free alternative, but graphical processes may run slower), and follow the instructions below

3. Dual Boot your computer (not recommended due to risk, but will lead to best results)



Installing VirtualBox


1. Go to VirtualBox download page, and download the version based on your operating system

2. Run the downloaded installation file, go though standard installation prompts.

3. Download Ubuntu 22.04 Desktop Image iso file. (You can reference previous videos for instructions)

4. Search your apps for VirtualBox, and run it. Click on the "New" Button

5. Configure Virtual Machine name and type

6. Configure Memory, generally leave as default.

7. Select VirtalBox Disk Image

8. Set size of hard disk. For this course, you can set it to 20 GB if you have a decent amount of free space on your computer. If you plan to use this OS outside the use of this course, you may want to consider adding more.

9. Go To Virtual Machine Settings (right click on VM name --> Settings)

10. Select the Disk Icon on the left under "Storage Devices", and then click on the disk icon on the right of "Optical Drive". Select "Choose Virtual Optical Disk File..." and select your Ubuntu ISO file downloaded earlier. Hit "OK".

11. Start your virtual machine by double clicking on it. You can refer to the previous video for setting up Ubuntu
